이미지 파일 처리 위치

현재 제공되는 이미지 크기 변경, 압축, 형식 변환, 프리셋 적용 기능은 사용자의 현재 브라우저 안에서 처리됩니다. 선택한 이미지는 외부 서버로 업로드되지 않고 Pixel Capsule 서버에 저장되지 않습니다.

처리 이미지 저장 여부

변환 결과는 브라우저 내부 URL로 표시됩니다. 페이지를 닫거나 새로고침하면 브라우저에 표시된 작업 결과는 사라질 수 있으며, 필요한 파일은 다운로드해 보관해야 합니다.

문의 시 수집 정보

GitHub Issues를 통해 문의하는 경우 사용자가 공개적으로 작성한 계정 이름, 문의 내용, 첨부 정보가 GitHub 정책에 따라 표시될 수 있습니다. 민감한 이미지나 개인정보가 포함된 파일은 공개 이슈에 업로드하지 않는 것을 권장합니다.

로그 및 분석 도구

현재 Google Analytics 코드는 삽입되어 있지 않습니다. 향후 분석 도구를 추가하는 경우 이 정책과 관련 안내를 갱신합니다.

쿠키와 로컬 저장소

사이트는 다크모드 설정처럼 사용자의 기기 안에 저장되는 로컬 설정을 사용할 수 있습니다. 이 정보는 이미지 파일 처리와 별개이며 서버로 업로드되지 않습니다.

Google AdSense와 제3자 광고

현재 실제 광고 코드는 삽입되어 있지 않고 광고 위치를 나타내는 placeholder만 표시합니다. 향후 Google AdSense 등 제3자 광고가 적용되면 광고 사업자가 쿠키 또는 유사 기술을 사용할 수 있으며, 맞춤 광고 설정은 사용자의 브라우저와 Google 광고 설정에서 관리할 수 있습니다.

개인정보 보관 기간

Pixel Capsule은 현재 이미지 파일을 서버에 저장하지 않습니다. 문의 기록은 GitHub Issues에 남을 수 있으며, 삭제가 필요한 경우 해당 플랫폼의 기능을 사용하거나 문의 페이지의 안내를 따라 요청해 주세요.

향후 기능 변경

서버 전송이 필요한 기능이 추가되는 경우 해당 기능 화면과 개인정보처리방침에서 별도로 안내합니다.

문의 방법

개인정보 관련 문의는 문의하기 페이지의 GitHub Issues 안내를 통해 접수할 수 있습니다.
