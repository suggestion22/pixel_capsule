문의하기

오류 제보, 기능 제안, 개인정보 또는 정책 문의가 있다면 아래 방법으로 알려주세요.

문의 방법

현재 Pixel Capsule은 별도 서버 문의 폼을 사용하지 않습니다. 오류 제보와 기능 제안은 GitHub Issues를 통해 접수합니다.

GitHub Issues 열기

보내주면 좋은 정보

  • 사용한 브라우저와 기기 종류
  • 이미지 형식과 대략적인 파일 크기
  • 오류가 발생한 도구 이름
  • 재현 단계와 화면에 표시된 오류 문구

개인정보 안내

문의에 이미지를 첨부할 필요는 없습니다. 민감한 이미지나 개인정보가 포함된 파일은 공개 이슈에 업로드하지 마세요.
