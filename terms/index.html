이용약관

Pixel Capsule을 사용하기 전에 아래 내용을 확인해 주세요. 최종 수정일: 2026년 7월 18일

서비스 범위

Pixel Capsule은 브라우저 기반 이미지 크기 변경, 압축, 형식 변환, 프리셋 선택 기능을 무료로 제공합니다. 향후 목표 용량 맞추기, 일괄 처리, 이미지 자르기 같은 기능이 추가될 수 있습니다.

사용자 책임

사용자는 자신이 권리를 가진 이미지 또는 사용 허가를 받은 이미지만 처리해야 합니다. 처리 결과의 최종 사용, 업로드, 배포 책임은 사용자에게 있습니다.

브라우저 처리와 기능 제한

현재 이미지 도구는 선택한 파일을 외부 서버로 업로드하지 않고 브라우저 안에서 처리합니다. 브라우저 환경, 이미지 크기, 파일 손상 여부, 기기 메모리에 따라 일부 처리가 실패할 수 있습니다.

결과 파일

변환 결과는 다운로드 전까지 브라우저에 임시로 표시됩니다. 페이지를 닫거나 새로고침하면 작업 결과가 사라질 수 있으므로 필요한 결과는 다운로드해 보관하세요.

광고

초기 버전에는 실제 광고 코드 대신 광고 위치 표시만 포함합니다. 광고가 적용되더라도 업로드, 변환, 다운로드, 아이디어 다시 보기 같은 조작 버튼과 혼동되는 위치에 배치하지 않습니다.

문의

오류 제보와 기능 제안은 문의하기 페이지를 통해 접수할 수 있습니다.
