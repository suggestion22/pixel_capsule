무료 이미지 형식 변환 도구

PNG, JPG, WebP 사이의 변환과 투명 배경 처리 옵션을 제공합니다.

형식 변환은 브라우저 안에서 이루어지며 결과 이미지에 워터마크를 넣지 않습니다.

이미지를 드래그하거나 클릭해서 선택하세요 JPG 변환 시 투명 배경은 선택한 배경색으로 채워집니다.

사용 방법

  1. 이미지를 선택합니다.
  2. 출력 형식, 품질, JPG 배경색을 설정합니다.
  3. 변환 결과를 확인하고 다운로드합니다.

주의사항

JPG는 투명도를 지원하지 않습니다. 투명 배경이 필요하면 PNG 또는 WebP를 선택해 주세요.
