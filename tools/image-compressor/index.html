Browser image tool

무료 이미지 압축 도구

품질 슬라이더로 용량과 화질의 균형을 찾고 결과 파일 크기를 바로 확인하세요.

압축은 브라우저의 Canvas API를 사용하며 현재 파일을 외부 서버로 전송하지 않습니다.

이미지를 드래그하거나 클릭해서 선택하세요 JPG, PNG, WebP 지원. PNG는 압축 효과가 제한될 수 있습니다.

Advertisement

사용 방법

  1. 이미지를 선택합니다.
  2. 출력 형식과 품질을 조절합니다.
  3. 압축 전후 용량을 비교한 뒤 다운로드합니다.

주의사항

낮은 품질로 저장한 이미지는 원래 선명도로 되돌리기 어렵기 때문에 원본 파일을 보관해 주세요.
