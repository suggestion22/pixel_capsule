목표 용량 도구의 역할

이 도구는 품질 값을 자동으로 여러 번 비교해 목표 파일 용량 이하에 가장 가까운 결과를 찾습니다. 직접 품질을 조절하고 싶다면 이미지 압축 도구를 사용하세요.

선택한 이미지는 외부 서버로 업로드되지 않고 현재 브라우저 안에서 처리됩니다. 정확히 같은 KB로 맞추는 기능이 아니라, 이미지 내용과 브라우저 인코딩 결과에 따라 가장 가까운 파일을 생성합니다.

지원 형식: JPG, PNG, WebP · 권장 파일 크기: 20MB 이하 · 고해상도 이미지는 기기 성능에 따라 처리되지 않을 수 있습니다.

  2. 2. 목표 용량 입력100KB, 500KB, 1MB 또는 직접 입력한 용량을 선택하세요.
  3. 3. 결과 확인목표 달성 여부와 실제 용량을 확인한 뒤 다운로드하세요.

목표 용량 압축 방식

JPG와 WebP는 품질 값을 낮추면 파일 용량이 줄어들 수 있습니다. 이 도구는 여러 품질 값을 빠르게 비교해 목표 용량 이하가 되는 가장 높은 품질을 찾습니다.

PNG는 JPG나 WebP처럼 품질 값을 낮추는 손실 압축이 제한적입니다. PNG 원본이 목표 용량에 도달하지 못한다면 WebP 또는 JPG 출력으로 비교하거나 이미지 크기를 먼저 줄여 보세요.

목표 용량은 보장값이 아닙니다. 이미지가 매우 복잡하거나 이미 강하게 압축되어 있으면 최저 품질에서도 목표 이하가 되지 않을 수 있습니다.

  • 페이지를 닫거나 새로고침하면 브라우저에 표시된 작업 결과는 사라질 수 있습니다.
  • 목표 용량에 도달하지 못하면 결과 정보에 미달성으로 표시됩니다.

자주 묻는 질문

정확히 500KB로 맞춰지나요?

정확히 같은 숫자로 맞추지는 않습니다. 브라우저가 생성한 결과 중 목표 용량 이하에 가장 가까운 파일을 선택합니다.

PNG도 목표 용량에 맞출 수 있나요?

PNG는 품질 값을 낮추는 방식이 제한적이라 목표 용량 달성이 어려울 수 있습니다. 투명 배경이 필요하다면 WebP 결과도 비교해 보세요.

화질은 얼마나 낮아지나요?

목표 용량이 작을수록 더 낮은 품질이 선택될 수 있습니다. 결과 정보에서 사용된 품질 값을 확인할 수 있습니다.

목표에 도달하지 못하면 어떻게 하나요?

이미지 크기 변경 도구로 해상도를 줄이거나 WebP/JPG 형식을 선택하면 더 작은 결과를 만들 가능성이 높아집니다.

이미지가 서버에 업로드되나요?

현재 제공되는 이미지 도구는 선택한 이미지를 외부 서버로 업로드하지 않고 브라우저 안에서 처리합니다.
