프리셋 도구의 역할

이 도구는 SNS, 블로그, 웹사이트에서 자주 사용하는 이미지 규격을 선택해 적용합니다. 원하는 픽셀 크기를 직접 입력하려면 이미지 크기 변경 도구를 사용하세요.

선택한 이미지는 외부 서버로 업로드되지 않고 현재 브라우저 안에서 처리됩니다. 페이지를 닫거나 새로고침하면 브라우저에 표시된 작업 결과는 사라질 수 있습니다.


플랫폼별 권장 이미지 규격은 변경될 수 있습니다. 중요한 작업에 사용할 경우 해당 플랫폼의 최신 안내도 함께 확인해 주세요.

이미지를 드래그하거나 클릭해서 선택하세요 프리셋을 선택하면 가로와 세로 값이 자동으로 적용됩니다.

지원 형식: JPG, PNG, WebP · 최대 파일 크기: 20MB · 현재 프리셋은 맞춤 크기 변경 방식으로 적용됩니다.

프리셋을 사용할 때 알아둘 점

플랫폼별 규격은 이미지가 표시되는 영역에 맞춰 권장되는 크기입니다. 같은 1:1 이미지라도 프로필, 게시물, 썸네일에서 실제 표시 방식은 다를 수 있습니다.

현재 프리셋 도구는 선택한 규격으로 이미지를 맞춰 크기를 변경합니다. 향후 자르기 기능이 추가되면 맞춤, 채우기, 자르기 방식을 선택할 수 있도록 확장할 예정입니다.

안전 영역은 잘림이나 UI 가림을 피하기 위해 핵심 피사체와 텍스트를 배치하는 중앙 영역을 뜻합니다. 스토리와 프로필 이미지는 특히 안전 영역을 고려하세요.

주의사항

알아두세요

플랫폼 권장 규격은 바뀔 수 있으므로 실제 업로드 전 최신 정책을 함께 확인하는 것이 좋습니다.

  • 프리셋 적용 후 이미지가 왜곡되지 않게 원본 비율 유지 상태를 확인하세요.
  • 서비스별 미리보기 자르기 방식은 시점과 화면 크기에 따라 달라질 수 있습니다.
  • 중요한 글자와 로고는 가장자리에서 충분히 떨어뜨려 배치하세요.

자주 묻는 질문

프리셋을 선택하면 이미지가 잘릴 수 있나요?

현재 도구는 선택한 크기로 리사이즈하는 방식입니다. 향후 자르기 모드가 추가되면 잘림 여부를 별도로 선택할 수 있게 할 예정입니다.

이미지가 찌그러지지 않게 하려면 어떻게 해야 하나요?

원본 비율 유지 옵션을 켜고, 필요한 경우 원본과 비슷한 비율의 프리셋을 선택하세요.

플랫폼별 규격은 변경될 수 있나요?

네. 각 플랫폼의 권장 규격은 변경될 수 있어 중요한 작업에는 최신 공식 안내도 함께 확인하는 것이 좋습니다.

같은 이미지로 여러 규격을 만들 수 있나요?

가능합니다. 하나의 이미지를 업로드한 뒤 다른 프리셋을 선택해 다시 변환하면 여러 규격을 만들 수 있습니다.

안전 영역이란 무엇인가요?

서비스 UI, 원형 자르기, 미리보기 잘림에 가려지지 않도록 중요한 내용을 배치하는 중앙 여유 영역입니다.
