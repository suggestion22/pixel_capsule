Browser image tool

무료 이미지 크기 변경 도구

원본 비율 유지, 백분율 조절, 결과 미리보기와 다운로드를 지원합니다.

선택한 이미지는 서버에 업로드하지 않고 사용자의 브라우저 안에서 처리됩니다.

이미지를 드래그하거나 클릭해서 선택하세요 JPG, PNG, WebP 지원. 파일은 브라우저 안에서 처리됩니다.

사용 방법

  1. 이미지를 선택합니다.
  2. 가로, 세로, 비율 유지, 백분율 값을 설정합니다.
  3. 변환 실행 후 결과를 다운로드합니다.

주의사항

작은 이미지를 원본보다 크게 키우면 파일만 커지고 선명도는 좋아지지 않을 수 있습니다.
